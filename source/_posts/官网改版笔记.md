## 插件使用记录

### [countUp.js](https://github.com/inorganik/countUp.js)
数字滚动变化插件。

用法：
```html
<h1 id="myTargetElement"></h1>
```
```js
    var options = {
        useEasing: true,
          useGrouping: true,
          separator: ',',
          decimal: '.',
    };
    var demo = new CountUp("myTargetElement", 0, 3831, 0, 3, options);

    demo.start();

```

### [swiper.js](https://github.com/nolimits4web/Swiper/tree/Swiper2)
为了IE的兼容性，使用swiper2。

[swiper2 API文档](http://2.swiper.com.cn/api/index.html)

用法：
```html
    <div class="swiper-container">
        <div class="swiper-wrapper">
            <div class="swiper-slide blue"> Slide1</div>
            <div class="swiper-slide red"> Slide2</div>
            <div class="swiper-slide yellow"> Slide3</div>
        </div>
        <div class="pagination"></div>
    </div>
```
```js
    $(function () {
        var mySwiper = new Swiper('.swiper-container', {
            pagination: '.pagination',
            paginationClickable: true,
        })
    });
```
```css
    .swiper-container {
        height: 300px;
        margin: 0 auto;
        position: relative;
        overflow: hidden;
        direction: ltr;
        backface-visibility: hidden;
        z-index: 1;
    }

    .swiper-slide {
        color: #fff;
        line-height: 300px;
        text-align: center;
        font-size: 50px;
    }

    .blue {
        background: #4390EE;
    }

    .red {
        background: #CA4040;
    }

    .yellow {
        background: #FF8604;
    }

    .pagination {
        position: absolute;
        z-index: 20;
        bottom: 10px;
        width: 100%;
        text-align: center;
    }

    .swiper-pagination-switch {
        display: inline-block;
        width: 8px;
        height: 8px;
        border-radius: 8px;
        background: #555;
        margin: 0 5px;
        opacity: 0.8;
        border: 1px solid #fff;
        cursor: pointer;
    }

    .swiper-active-switch {
        background: #fff;
    }
```
